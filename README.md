# LABELS
Here's the list of all labels I use in all my repos (for consistency).\
*Note some repos don't use all these labels.*

## Labels for issue or pull request categories

| Name | Color | Description |
|---|---|---|
| bug | ``#e11616`` | Unexpected problem or unintended behaviour found |
| compatibility | ``#e1413c`` | States incompatible features or impossible runs with some platforms or OS systems |
| enhancement | ``#00fffe`` | Request for new feature |
| maintenance | ``#49bddc`` | Maintenance required on some features |
| performance | ``#90ffeb`` | Request for optimisation |

## Labels for extra information

| Name | Color | Description |
|---|---|---|
| need deep analysis | ``#ff00ff`` | Asks for further analysis on the issue or pull request |
| need reproduction | ``#ffc7ff`` | Asks for reproducibility of the issue |
| missing info | ``#ff5ca3`` | Cannot accept pull request or solve issue due to missing information |
| need review | ``#6200da`` | Asks for review on the issue or pull request (or changes made) |
| need testing | ``#000ede`` | Need to test changes or pull request |

## Labels for high priority

| Name | Color | Description |
|---|---|---|
| urgent | ``#ff4900`` | Give maximum priority to pull request or issue |
| critical | ``#ff6921`` | Critical bug or security problem found |
| security | ``#ffa321`` | Issue or pull request related to security issues or failures | 

## Labels for status (progression)

| Name | Color | Description |
|---|---|---|
| in progress | ``#ffffbf`` | Working on this issue or pull request |
| tested | ``#488963`` | States changes or pull request that passes tests |
| fixed | ``#2cc863`` | Fixed bug or compatibility issue | 
| added | ``#0bd278`` | New feature added |
| rejected | ``#171717`` | Rejected issue or pull request |
| duplicate | ``#a1a1a1`` | Similar issue or pull request |

## Labels for status (releases)

| Name | Color | Description |
|---|---|---|
| wait for release | ``#ffe91e`` | Waits for merging into release branch |
| released | ``#80c500`` | Released in an official version |
| beta/dev | ``14ff00`` | Released in a beta or dev or other instable version |

## Labels for "wait some people"

| Name | Color | Description |
|---|---|---|
| good first issue | ``#c170ff`` | Sets the issue for first-time contributors |
| help wanted | ``#c1baff`` | Waits for people to work on it |
| on hold | ``9100ab`` | Currently not working on but other interested contributors can work on it (assign yourself) |
